# Autobuses del Río

Sitio web oficial de Autobuses del Río, empresa de transporte con más de 50 años de experiencia en Toledo, España.

## 🚌 Acerca de Nosotros

Autobuses del Río es una empresa familiar de transporte que ofrece servicios de calidad en toda España y Portugal. Con una flota de autobuses modernos y un equipo experimentado, nos especializamos en:

- **Rutas Escolares**: Transporte seguro y confiable para estudiantes
- **Contratación Privada**: Servicios para clubes deportivos y grupos
- **Excursiones**: Viajes turísticos y culturales
- **Bodas y Eventos**: Transporte elegante para ocasiones especiales
- **Administración Pública**: Servicios para instituciones gubernamentales

## 🛠️ Tecnologías Utilizadas

- **Next.js 14** - Framework de React para aplicaciones web
- **TypeScript** - Tipado estático para JavaScript
- **Tailwind CSS** - Framework de CSS utilitario
- **Shadcn/ui** - Componentes de interfaz de usuario
- **Radix UI** - Componentes accesibles y sin estilos
- **Framer Motion** - Animaciones fluidas

## 🚀 Instalación y Desarrollo

### Prerrequisitos
- Node.js 18 o superior
- Yarn (recomendado) o npm

### Instalación
```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/autobuses-del-rio.git

# Navegar al directorio
cd autobuses-del-rio

# Instalar dependencias
yarn install

# Ejecutar en modo desarrollo
yarn dev
```

### Scripts Disponibles
```bash
yarn dev          # Servidor de desarrollo
yarn build        # Construir para producción
yarn start        # Servidor de producción
yarn lint         # Linting del código
```
